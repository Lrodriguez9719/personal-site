# Capturas de pantalla — Guía ayuda Binance

Coloca aquí las capturas de pantalla del proceso. Mientras un archivo no exista,
la página mostrará automáticamente un recuadro de "captura pendiente".

Nombres de archivo esperados (PNG o JPG, deben llamarse exactamente así):

| Archivo        | Paso                                            |
| -------------- | ----------------------------------------------- |
| `paso1.png`    | Crear cuenta / descargar la app de Binance      |
| `paso2.png`    | Verificación de identidad (eKYC)                |
| `paso3.png`    | Verificación de domicilio (POA)                |
| `paso4.png`    | Notificación / espera del depósito              |
| `paso5.png`    | Reclamar el cupón en la Zona de Recompensas     |

Recomendación: imágenes verticales (de celular) de ancho ~750px o más.
Si usas `.jpg` en lugar de `.png`, actualiza la extensión en `pages/ayuda_binance.html`.
